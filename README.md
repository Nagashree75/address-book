# address-book
This project is designed to strengthen core C programming concepts such as structures, pointers, file handling, and menu-driven programs.

📘 Address Book Management System in C
A simple console-based Address Book application developed using the C programming language. This project allows users to store, manage, and organize contact details efficiently using file handling.
🚀 Features
➕ Add new contacts
📋 View all contacts
🔍 Search contacts by name or phone number
✏️ Edit existing contact details
❌ Delete contacts
💾 Save and load contacts using file handling
🛠️ Technologies Used
C Programming Language
Standard Libraries:
stdio.h
string.h
stdlib.h
🎯 Project Objective
The main goal of this project is to practice and understand:
Structures in C
File handling
Functions
Menu-driven programming
Basic data management

⚙️ How to Run
Clone the repository:

git clone https://github.com/your-username/address-book.git
Navigate to the project folder:

cd address-book
Compile the program:

gcc address_book.c -o address_book
Run the program:

./address_book

1. Add Contact
